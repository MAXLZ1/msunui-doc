进度条 Progress

用来显示时间进度。
使用示例
进度条code
构造器参数说明(new Progress(obj))
参数 参数类型 参数值 备注
elem String/JQueryDOM - 进度条容器
max Number 默认100 进度条最大值
value Number 默认0 进度条进度值
showText Boolean 默认false 是否显示进度值
color String 默认"#3896fe" 配置进度条颜色
开放接口
方法名 参数 返回值 备注
reload(value,max)

value:Number,进度条的值

max:Number,进度条最大值,若进度条最大值未发生变化可不选

 返回当前实例 刷新进度条
