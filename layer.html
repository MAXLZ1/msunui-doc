弹层 Layer

弹层主要用于信息提示，替代了window.alert/window.confirm
开放接口
方法名 参数类型 返回值 用途 效果展示
open(obj)

Object

参数结构
{ shade:true, // 是否显示遮罩，默认true shadeType:0, // 遮罩类型，默认为0,0表示深色遮罩，1表示白色透明遮罩 move:false, // 是否可移动，默认false close:true, // 是否可关闭，默认true width:500, // 弹层宽度,默认500 height:450, // 弹层高度,默认450 position:[], // 定位数组，依次为上、右、下、左 title:"标题" // 弹层标题 content:"", // 内容,content与url二选一 url:"" // 页面路径 }
返回弹层id 弹出模态框 open openTable openSelect
alert(obj)

Object

参数结构
{ width:250, // 弹层宽度，默认250 height:165, // 弹层高度，默认165 title:"提示", // 弹层标题,默认“提示” content:"", // 内容 callback:function(){} // 点击确定回调函数 }
返回弹层id 弹出提示框，作用与window.alert一致 alert
confirm(obj)

Object

参数结构
{ width:250, // 弹层宽度，默认250 height:165, // 弹层高度，默认165 title:"提示", // 弹层标题,默认“提示” content:"", // 内容 callback:function(data){} // 回调函数，data为点击的是确定还是取消 }
返回弹层id 弹出对话框，作用与window.confirm一致 confirm
msg(obj)

Object

参数结构
{ content:"", // 内容 time:2000, // 显示多长时间,单位ms，默认2s callback:function(){} // 消失回调函数 }
返回弹层id 自动消失的提示框 msg
close(id) Number - 关闭弹层
closeAll() - - 关闭所有弹层
代码示例
参数结构
var layer = new Layer(); layer.open({ title:"百度", height:$(window).height()*0.7, width:$(window).width()*0.6, url:"http://www.baidu.com" }); layer.open({ title:"百度", content:$("#tabledemo") }); layer.alert({ content:"这是一个alert框" }); layer.confirm({ content:"这是一个confirm框" }); layer.msg({ content:"我将在2s后消失！" });
