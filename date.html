日期/时间 DatePicker

提供多类型日期选择，如日、周、月、季、年。日期组件对于周的计算采用两种算法：1.ISO8601:使用每年的的第一个周四所在的周为第一周；2.NORMAL：使用每年的1月1日所在的周为第一周。

开放接口
方法名 参数 说明
init - 初始化，一般不需要使用。
on

eventType:事件类型，参考事件类型

callback:回调函数

 注册事件
事件类型
事件名 事件说明 回调说明
done 时间选择完毕事件，点击日期、确定、现在、清空均可触发，即一切可确定目标节点的日期值的操作均触发done事件

callback(obj)

obj:被选中的值
参数说明
参数 参数类型 参数值 说明
elem String/JQueryDom - 日期触发dom
type String

可选值

1.date:日类型，默认值

2.week:周类型

3.month:月类型

4.quarter:季类型

5.year:年类型

6.time:时间类型

7.datetime:日期时间类型

 日期组件类型
format String

date默认:yyyy-MM-dd

week默认:yyyy-ww

quarter默认:yyyy-qq

month默认:yyyy-MM

year默认:yyyy

time默认:HH:mm:ss

date默认:yyyy-MM-dd HH:mm:ss

关于format使用参考format格式化说明

 日期格式化模板
value String 默认为当前时间 日期组件的值
weekAlgorithm String

可选值：

1.ISO8601:每年的第一个周四所在的周为第一周

2.NORMAL:每年的1月1日所在的那周为第一周

 周算法，日期类型为week时有效
firstDayOfWeek Number

可选值：

0：周天

1：周一

 设置日期组件第一周从哪天起
format格式化说明
格式符 说明
yyyy 年份，不足四位自动补零
qq 季度，不足两位自动补零
q 季度，不补零
MM 月份，不足两位自动补零
M 月份，不补零
ww 周，不足两位自动补零
w 周，不补零
dd 日，不足两位自动补零
d 日，不补零
HH 小时，不足两位自动补零
H 小时，不补零
mm 分钟，不足两位自动补零
m 分钟，不补零
ss 秒，不足两位自动补零
s 秒，不补零
