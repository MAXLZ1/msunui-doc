表单

表单组件中的多选框、单选框、下拉框均自动渲染，无需new Form。单若重新渲染或使用一些Form的方法，必须new一个Form。注：在表单元素中添加ignore-render可阻止元素的渲染

addValidationRule(obj)

obj:要添加的验证对象，两种添加方式：

1：方法方式

方法判断
form.addValidationRule({ name:function(obj){ // obj为当前表单的值 if(/^\d+\d+\d$/.test(obj)){ return "不能全为数字"; } } });

2.正则方式

正则方式
form.addValidationRule({ name：{ regExp:/^\d+\d+\d$/, hint:"不能全为数字" } });

var form2 = new Form({ elem:"#selectForm" }); form2.on("select(#selectDemo)",function (obj) { console.log(obj); $("#selectDemoText").text(obj.text); }); form2.on("radio(input[name='sex1'])",function (obj) { console.log(obj); $("#radioDemoText").text(obj.text); }); form2.on("checkbox(input[name='sports1'])",function (obj) { console.log(obj); var text = $("#checkboxDemoText").text().replace(/\s+/,''), arr = text ? text.split(',') : []; if(obj.checked){ arr.push(obj.text); }else{ $.each(arr, function (index,item) { if(item === obj.text){ arr.splice(index, 1); return false; } }); } $("#checkboxDemoText").text(arr.join(',')); });
表单验证

表单验证只需在需要验证的元素上添加data-validate属性即可，多个验证规则使用|分隔，如<input data-validate="require|integer"/>。关于验证规则，Form组件内置多种验证规则，具体参考正则库,也可使用addValidationRule添加自定义验证规则。
