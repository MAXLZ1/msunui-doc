步骤条 Steps

显示事件步骤信息。
使用示例
steps code
构造器参数说明(new Steps(obj))
参数 参数类型 参数值 备注
elem String/JQueryDOM - 进度条容器
data Array 参考data参数说明 步骤条数据信息
space Number/String 可设置具体宽度及百分比 设置间距
direction String 可选值：["horizontal","vertical"],默认"horizontal" 设置步骤条方向，该参数只在type="line"情况下生效
nodeType String 可选值：["number","custom"],默认"number" 设置节点类型：number为数字；custom自定义
type String 可选值：["line","block"],默认"line" 设置步骤条类型
data参数说明
属性 属性类型 属性值 备注
title String - 步骤条标题
desc String - 步骤条描述信息
status Number 可选值：[0,1],0表示待完成，1表示已完成，默认为0 当前状态
icon String iconClass 设置节点显示图标，在nodeType="custom"时生效
