分页 Page

开放接口
方法名 参数 说明
init - page初始化,一般不需要。new的过程中已经初始化
on

eventType:事件类型,"skip"(跳转页面、更换分页条数均会触发)

callback:回调函数

 page绑定事件
参数说明
参数名 参数类型 参数值 说明
elem String/JQueryDom dom选择器或JQuery对象 选择分页容器
count Number - 数据总数
limit Number 默认为10 每页显示数量，该值取决于limits
limits Array 默认值为[10,20,50,100,150,200] 每页显示条数数组
curr Number 默认1 当前页
showPageNum Number 默认为7，超过7页显示省略号 可显示页码数
